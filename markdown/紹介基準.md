# 腎臓内科への紹介基準（例）

1. CKD（慢性腎臓病）の進行が疑われる場合
   - eGFR < 30 mL/min/1.73m²（G4以下）は原則紹介
   - eGFR 30~60で以下のいずれかを満たす場合
     - 尿蛋白UPCR ≥ 0.5 g/gCr または ACR ≥ 300 mg/gCr
     - eGFRが1年で5以上低下、もしくは3年で10以上低下
     - 難治性高血圧（3剤以上でもコントロール困難）

2. 尿蛋白・血尿などから腎炎が疑われる場合
   - 尿蛋白が持続的に陽性（定性2+以上）で持続
   - 尿蛋白＋血尿（特に変形赤血球や円柱）が持続
   - ネフローゼ症候群が疑われる（Alb低値、浮腫、蛋白尿大量など）

3. 急性腎障害が疑われる場合
   - 数日〜数週でCrが急増、eGFRが急激に低下
   - 尿量減少、体重増加、浮腫、呼吸苦などを伴う

4. 電解質異常を伴う場合
   - 高カリウム血症（K ≥ 6.0）を認める
   - 重度の代謝性アシドーシスが疑われる

5. 原因不明の腎機能低下
   - 明らかな誘因がなく、徐々に腎機能が悪化している症例

6. 主治医が「腎臓内科に相談したい」と感じる場合
   - 紹介基準に明確に当てはまらなくても、相談目的の紹介は歓迎とする
