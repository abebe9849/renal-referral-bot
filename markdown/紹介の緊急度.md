# 腎臓内科受診の緊急度の目安

1. 当日〜数日以内の受診が望ましい
   - 急激な腎機能悪化（AKI）の疑い
   - 高カリウム血症（K ≥ 6.0）を認める
   - 急速な体重増加や高度浮腫、呼吸苦など心不全増悪が疑われる
   - 尿量の著明な減少（乏尿〜無尿）
   - ネフローゼ症候群が疑われる

2. 1〜2週間以内の受診が望ましい
   - CKD G4（eGFR < 30）で新規紹介
   - CKD G3b（eGFR 30–44）で尿蛋白高度、または進行が疑われる
   - 高度の尿蛋白血尿が持続しているが、急激な変化はない

3. 通常の予約受診で問題ない
   - CKD G3a（eGFR 45–59）で尿蛋白が軽度、進行も緩徐
   - 安定した糖尿病性腎症や高血圧性腎障害で、治療方針確認目的
   - 主治医によるフォローで大きな問題はないが、一度方針相談を希望する場合
